# Dhanushka Weerakoon — Portfolio

Personal portfolio website of **Dhanushka Weerakoon**, Mechanical Engineering (Hons.) graduate of the University of Moratuwa.

🔗 **Live site:** https://dhanushkadcw.github.io/Dhanushka_Weerakoon/

## Built with
- Single-page, responsive HTML / CSS / vanilla JavaScript (no frameworks, no build step)
- Scroll-reveal animations, filterable project gallery, typing effect
- Hosted free on **GitHub Pages**

## Structure
```
index.html                 # the whole site (HTML + CSS + JS)
assets/
  Dhanushka_Weerakoon_CV.pdf
  img/                     # project & profile images
```

## Editing
Everything lives in `index.html`. Project entries are defined in the `PROJECTS` array near the bottom — edit text, tags, category or image there and the gallery updates automatically.
